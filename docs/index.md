#### simpleCountdown

```python
def simpleCountdown(seconds, message="Waiting...")
```

Simple countdown with message

**Arguments**:

- `seconds` _int_ - Duration of countdown in seconds
- `message` _str, optional_ - Message of countdown. Defaults to "Waiting...".
